HTMLAreaElement.href

The HTMLAreaElement.href property is a stringifier that returns a USVString containing the whole URL, and allows the href to be updated.

Syntax

// Getter
string = area.href;
// Setter
area.href = string;

Examples

// An <area id="myArea" href="https://developer.mozilla.org/en-US/HTMLAreaElement"> element is in the document
const area = document.getElementById("myArea");
area.href; // returns 'https://developer.mozilla.org/en-US/HTMLAreaElement'
