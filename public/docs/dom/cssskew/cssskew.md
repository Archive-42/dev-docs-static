# CSSSkew()

**This is an [experimental technology](https://developer.mozilla.org/en-US/docs/MDN/Guidelines/Conventions_definitions#experimental)**  
Check the [Browser compatibility table](#browser_compatibility) carefully before using this in production.

**Draft**

This page is not complete.

The `CSSSkew()` constructor creates a new [`CSSSkew`](../cssskew) object which represents the `skew()` value of the individual [`transform`](https://developer.mozilla.org/en-US/docs/Web/CSS/transform) property in CSS.

## Syntax

    var CSSSkew = new CSSSkew(ax, ay);

### Parameters

[`ax`](ax)  
A value for the `ax` (x-axis) angle of the [`CSSSkew`](../cssskew) object to be constructed. This must be a [`CSSNumericValue`](../cssnumericvalue).

[`ay`](ay)  
A value for the `ay` (y-axis) angle of the [`CSSSkew`](../cssskew) object to be constructed. This must be a [`CSSNumericValue`](../cssnumericvalue).
