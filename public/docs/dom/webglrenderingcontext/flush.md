WebGLRenderingContext.flush()
=============================

The `WebGLRenderingContext.flush()` method of the [WebGL API](../webgl_api) empties different buffer commands, causing all commands to be executed as quickly as possible.

Syntax
------

    void gl.flush();

### Parameters

None.

### Return value

None.
