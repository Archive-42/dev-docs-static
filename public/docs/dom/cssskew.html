The CSSSkew interface of the CSS_Object_Model#css_typed_object_model is part of the CSSTransformValue interface. It represents the skew() value of the individual transform property in CSS.

Constructor

CSSSkew.CSSSkew()
Creates a new CSSSkew object.

Properties

ax
Returns or sets the x-axis value.
ay
Returns or sets the y-axis value.
