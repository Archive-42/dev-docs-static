WebSocket.url
=============

The `WebSocket.url` read-only property returns the absolute URL of the [`WebSocket`](../websocket) as resolved by the constructor.

Syntax
------

    var url = aWebSocket.url;

Value
-----

A [`DOMString`](../domstring).
