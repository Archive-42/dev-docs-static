# DeviceMotionEventRotationRate

**Experimental**

**This is an [experimental technology](https://developer.mozilla.org/en-US/docs/MDN/Guidelines/Conventions_definitions#experimental)**  
Check the [Browser compatibility table](#browser_compatibility) carefully before using this in production.

A `DeviceMotionEventRotationRate` object provides information about the rate at which the device is rotating around all three axes.

## Properties

[`DeviceMotionEventRotationRate.alpha`](devicemotioneventrotationrate/alpha) <span class="badge inline readonly">Read only </span>  
The amount of rotation around the Z axis, in degrees per second.

[`DeviceMotionEventRotationRate.beta`](devicemotioneventrotationrate/beta) <span class="badge inline readonly">Read only </span>  
The amount of rotation around the X axis, in degrees per second.

[`DeviceMotionEventRotationRate.gamma`](devicemotioneventrotationrate/gamma) <span class="badge inline readonly">Read only </span>  
The amount of rotation around the Y axis, in degrees per second.
