Sensor.start()
==============

The `start` method activates one of the sensors based on `Sensor`.

Syntax
------

    Sensor.start()

### Parameters

None.

### Return value

[`undefined`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/undefined)
