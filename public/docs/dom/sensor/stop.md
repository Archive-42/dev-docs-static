Sensor.stop()
=============

The `stop` method of the [`Sensor`](../sensor) interface deactivates the current sensor.

Syntax
------

    Sensor.stop()

### Parameters

None.

Return value

[`undefined`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/undefined)
