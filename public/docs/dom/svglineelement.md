SVGLineElement
==============

The `SVGLineElement` interface provides access to the properties of [`<line>`](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/line) elements, as well as methods to manipulate them.

Properties
----------

*This interface also inherits properties from its parent, [`SVGGeometryElement`](svggeometryelement).*

 <span class="page-not-created">`SVGLineElement.x1`</span> <span class="badge inline readonly">Read only </span>   
Returns an [`SVGAnimatedLength`](svganimatedlength) that corresponds to attribute `x1` on the given [`<line>`](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/line) element.

 <span class="page-not-created">`SVGLineElement.y1`</span> <span class="badge inline readonly">Read only </span>   
Returns an [`SVGAnimatedLength`](svganimatedlength) that corresponds to attribute `y1` on the given [`<line>`](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/line) element.

 <span class="page-not-created">`SVGLineElement.x2`</span> <span class="badge inline readonly">Read only </span>   
Returns an [`SVGAnimatedLength`](svganimatedlength) that corresponds to attribute `x2` on the given [`<line>`](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/line) element.

 <span class="page-not-created">`SVGLineElement.y2`</span> <span class="badge inline readonly">Read only </span>   
Returns an [`SVGAnimatedLength`](svganimatedlength) that corresponds to attribute `y2` on the given [`<line>`](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/line) element.

Methods
-------

*This interface doesn't implement any specific methods, but inherits methods from its parent interface, [`SVGGeometryElement`](svggeometryelement).*
