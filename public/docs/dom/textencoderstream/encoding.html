TextEncoderStream.encoding

The encoding read-only property of the TextEncoderStream interface returns a DOMString containing the name of the encoding algorithm used by the current TextEncoderStream object.

Syntax

var encoding = TextEncoderStream.encoding;

Value

A DOMString containing utf-8 encoded data.

Examples

The following example demonstrates how to return encoding from a TextEncoderStream object.

stream = new TextEncoderStream();
console.log(stream.encoding);
