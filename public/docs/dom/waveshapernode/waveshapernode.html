WaveShaperNode.WaveShaperNode()

The WaveShaperNode() constructor of the Web Audio API creates a new WaveShaperNode object which is an AudioNode that represents a non-linear distorter.

Syntax

var waveShaperNode = new WaveShaperNode(context, options)

Parameters

Inherits parameters from the AudioNodeOptions dictionary.

context
A reference to an AudioContext.
options Optional
Options are as follows:
  • curve: The shaping curve used for the waveshaping effect. The input signal is nominally within the range [-1;1].
  • oversample: Specifies what type of oversampling (if any) should be used when applying the shaping curve. Valid values are 'none', '2x', or '4x'. The default is 'none'.

Return value

A new WaveShaperNode object instance.
