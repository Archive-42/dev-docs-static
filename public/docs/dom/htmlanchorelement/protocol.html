HTMLAnchorElement.protocol

The HTMLAnchorElement.protocol property is a USVString representing the protocol scheme of the URL, including the final ':'.

Syntax

// Getter
string = anchor.protocol;
// Setter
anchor.protocol = string;

Examples

Getting the protocol of an anchor link

// An <a id="myAnchor" href="https://developer.mozilla.org/en-US/HTMLAnchorElement"> element is in the document
const anchor = document.getElementById("myAnchor");
anchor.protocol; // returns 'https:'
