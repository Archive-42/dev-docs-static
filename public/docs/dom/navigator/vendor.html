Navigator.vendor

The value of the Navigator vendor property is always either "Google Inc.", "Apple Computer, Inc.", or (in Firefox) the empty string.

Syntax

venString = window.navigator.vendor

Value

  • Either "Google Inc.", "Apple Computer, Inc.", or (in Firefox) the empty string.
