HTMLTableElement.tBodies

The HTMLTableElement.tBodies read-only property returns a live HTMLCollection of the bodies in a <table>.

Although the property is read-only, the returned object is live and allows the modification of its content.

The collection returned includes implicit <tbody> elements. For example:

<table>
  <tr>
    <td>cell one</td>
  </tr>
</table>

The HTML DOM generated from the above HTML will have a <tbody> element even though the tags are not included in the source HTML.

Syntax

HTMLCollectionObject = table.tBodies

Example

This snippet gets the number of bodies in a table.

mytable.tBodies.length;
