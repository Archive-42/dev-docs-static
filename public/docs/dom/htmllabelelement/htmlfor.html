HTMLLabelElement.htmlFor

The HTMLLabelElement.htmlFor property reflects the value of the for content property. That means that this script-accessible property is used to set and read the value of the content property for, which is the ID of the label's associated control element.

Syntax

controlId = HTMLLabelElement.htmlFor

HTMLLabelElement.htmlFor = newId

Value

A DOMString which contains the ID string of the element which is associated with the control.

If this property has a value, the HTMLLabelElement.control property must refer to the same control.
