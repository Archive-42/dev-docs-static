CSSUnparsedValue.length

Experimental

The length read-only property of the CSSUnparsedValue interface returns the number of items in the object.

Syntax

var length = CSSUnparsedValue.length;

Value

An integer.

Examples

In this example we employ the CSSUnparsedValue.CSSUnparsedValue() constructor, then query the length:

let values = new CSSUnparsedValue( ['1em', '#445566', '-45px'] );

console.log( values.length ) // 3
