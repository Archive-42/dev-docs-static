XRInputSourceEventInit

Secure context

This feature is available only in secure contexts (HTTPS), in some or all supporting browsers.

The XRInputSourceEventInit dictionary is used when calling the XRInputSourceEvent() constructor to provide configuration options for the newly-created XRInputSourceEvent object to take on.

Properties

The XRInputSourceEventInit dictionary inherits properties from the EventInit dictionary. It also offers the following:

frame
An XRFrame object representing the event frame during which the event took place. This event is not associated with the animation process, and has no viewer information contained within it.
inputSource
An XRInputSource object representing the input device from which the event is being sent.
