The sub() method of the CSSNumericValue interface subtracts a supplied number from the CSSNumericValue.

Syntax

var cssMathSum = CSSNumericValue.sub(number);

Parameters

number
Either a Number or a CSSMathSum

Return value

A CSSMathSum

Exceptions

TypeError
Indicates that an invalid type was passed to the method.

Examples

let mathSum = CSS.px("23").sum(CSS.percent("4")).sum(CSS.cm("3")).sum(CSS.in("9"));
// Prints "calc(23px - 4% - 3cm - 9in)"
console.log(mathSum.toString());
