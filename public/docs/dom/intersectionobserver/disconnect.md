IntersectionObserver.disconnect()
=================================

The [`IntersectionObserver`](../intersectionobserver) method `disconnect()` stops watching all of its target elements for visibility changes.

Syntax
------

    intersectionObserver.disconnect();

### Parameters

None.

### Return value

`undefined`.
