Experimental

Check the Browser compatibility table carefully before using this in production.

The lang property ofthe NDEFRecord interface is USVString containing the language tag of the record contents, if it is a vailable.

The record might be missing a language tag, for example, if the recorded information is not locale-specific.

Syntax

NDEFRecord.lang

Value

A USVString.
