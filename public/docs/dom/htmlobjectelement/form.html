HTMLObjectElement.form

The form read-only property of the HTMLObjectElement interface returns a HTMLFormElement representing the object element's form owner, or null if there isn't one.

Syntax

var HTMLFormElement = HTMLObjectElement.form;

Value

A HTMLFormElement.
