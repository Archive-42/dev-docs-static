BluetoothRemoteGATTDescriptor.characteristic

Experimental

This is an experimental technology
Check the Browser compatibility table carefully before using this in production.

The BluetoothRemoteGATTDescriptor.characteristic read-only property returns the BluetoothRemoteGATTCharacteristic this descriptor belongs to.

Syntax

var characteristic = BluetoothRemoteGATTCharacteristic.characteristic

Returns

An instance of BluetoothRemoteGATTCharacteristic.
