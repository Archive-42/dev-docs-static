MediaKeySession.update()

The MediaKeySession.update() method loads messages and licenses to the CDM, and then returns a Promise .

Syntax

mediaKeySession.update(response).then(function() { ... });

Parameters

response
An instance of type BufferSource.

Return value

A Promise.
