# DedicatedWorkerGlobalScope.onmessageerror

The `onmessageerror` event handler of the [`DedicatedWorkerGlobalScope`](../dedicatedworkerglobalscope) interface is an [`EventListener`](../eventlistener), called whenever an [`MessageEvent`](../messageevent) of type `messageerror` is fired on the workerâ€”that is, when it receives a message that cannot be [deserialized](https://developer.mozilla.org/en-US/docs/Glossary/Deserialization).

**Note:** This feature is available in [Web Workers](../web_workers_api).

## Syntax

    onmessageerror = function() { ... };
