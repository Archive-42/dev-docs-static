BackgroundFetchEvent.registration

The registration read-only property of the BackgroundFetchEvent interface returns a BackgroundFetchRegistration object.

Syntax

let registration = BackgroundFetchEvent.registration;

Value

A BackgroundFetchRegistration.

Examples

In this example, if the user clicks on the user interface displaying the download progress, this fires the onbackgroundfetchclick event. The current BackgroundFetchRegistration is returned by calling event.registration.

addEventListener('backgroundfetchclick', (event) => {
  console.log(event.registration);
});
