RTCIceCandidateStats.url


The RTCIceCandidateStats dictionary's url property specifies the URL of the ICE server from which the described candidate was obtained. This property is only available for local candidates.

Syntax

url = rtcIceCandidateStats.url;

Value

A DOMString specifying the URL of the ICE server from which the candidate described by the RTCIceCandidateStats was obtained. This is the same URL that would be received in the icecandidate event's url property.

Note: This property does not exist for remote candidates.
