The appendBuffer() method of the SourceBuffer interface appends media segment data from an ArrayBuffer or ArrayBufferView object to the SourceBuffer.

Syntax

sourceBuffer.appendBuffer(source);

Parameters

source
A BufferSource (that is, either an ArrayBufferView or ArrayBuffer) which contains the media segment data you want to add to the SourceBuffer.

Return value

undefined.

Exceptions

None.
