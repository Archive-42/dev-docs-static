# DOMParser()

The `DOMParser()` constructor creates a new `DOMParser` object.

## Syntax

    const parser = new DOMParser();

### Parameters

None.

### Return value

A new `DOMParser` object. This object can be used to parse the text of a document using the `parseFromString()` method.
