TrackEvent.track

The read-only track property of the TrackEvent interface specifies the media track object to which the event applies. This will be an AudioTrack, VideoTrack, or TextTrack object.

Syntax

track = TrackEvent.track;

Value

An object which is one of the types AudioTrack, VideoTrack, or TextTrack, depending on the type of media represented by the track. This identifies the track to which the event applies.
