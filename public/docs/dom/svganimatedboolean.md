SVGAnimatedBoolean
==================

SVG animated boolean interface
------------------------------

The `SVGAnimatedBoolean` interface is used for attributes of type boolean which can be animated.

Methods

The `SVGAnimatedBoolean` interface do not provide any specific methods.
