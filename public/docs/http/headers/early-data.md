Early-Data
==========

**This is an [experimental technology](https://developer.mozilla.org/en-US/docs/MDN/Contribute/Guidelines/Conventions_definitions#Experimental)**  
Check the [Browser compatibility table](#Browser_compatibility) carefully before using this in production.

The `Early-Data` header is set by an intermediary to indicate that the request has been conveyed in [TLS early data](https://developer.mozilla.org/en-US/docs/Web/Security/Transport_Layer_Security#TLS_1.3), and also indicates that the intermediary understands the [`425 (Too Early)`](../status/425) status code.

The `Early-Data` header is **not** set by the originator of the request (i.e., a browser).

Syntax
------

    Early-Data: 1

Examples
--------

    GET /resource HTTP/1.0
    Host: example.com
    Early-Data: 1
