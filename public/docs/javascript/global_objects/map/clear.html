Map.prototype.clear()

The clear() method removes all elements from a Map object.

Syntax

clear()

Return value

undefined.

Examples

Using clear()

var myMap = new Map();
myMap.set('bar', 'baz');
myMap.set(1, 'foo');

myMap.size;       // 2
myMap.has('bar'); // true

myMap.clear();

myMap.size;       // 0
myMap.has('bar')  // false
