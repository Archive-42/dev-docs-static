# CSS Scroll Snap

**CSS Scroll Snap** is a module of CSS that introduces scroll snap positions, which enforce the scroll positions that a [scroll container](https://developer.mozilla.org/en-US/docs/Glossary/Scroll_container)â€™s [scrollport](https://developer.mozilla.org/en-US/docs/Glossary/Scrollport) may end at after a scrolling operation has completed.
