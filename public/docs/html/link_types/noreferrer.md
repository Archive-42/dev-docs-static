Link types: noreferrer
======================

The `noreferrer` keyword for the `rel` attribute of the [`<a>`](../element/a), [`<area>`](../element/area), and [`<form>`](../element/form) elements instructs the browser, when navigating to the target resource, to omit the [`Referer`](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Referer) header and otherwise leak no referrer information â€” and additionally to behave as if the `noopener` keyword were also specified.
